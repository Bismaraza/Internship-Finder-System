ONLINE INTERNSHIP PORTAL SYSTEM - Java Swing + MySQL + JDBC

1. Open XAMPP and start Apache + MySQL.
2. Open phpMyAdmin.
3. Import database/internship_finder.sql.
4. Open Eclipse IDE.
5. File > Import > Existing Projects into Workspace.
6. Select InternshipFinder folder.
7. Add MySQL Connector/J to Build Path:
   - Put mysql-connector-j.jar inside lib/
   - Right click project > Build Path > Configure Build Path > Libraries > Add JARs
8. Run src/main/Main.java.
